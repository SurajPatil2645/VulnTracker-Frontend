# 🖥️ Vulnerability Monitoring Dashboard – Frontend

## Project Overview and Objectives

This frontend module is part of the **Automated Vulnerability Scraping and Reporting System** developed under a Project-Based Learning (PBL) initiative.

The objective of the frontend is to provide a **user-friendly web interface** that allows users to view, analyze, and download vulnerability reports generated by the backend system.

The frontend focuses on usability, responsiveness, and clear visualization of security vulnerability data.

---

## Features Implemented

- User login and authentication pages
- Dashboard displaying recent vulnerabilities
- Severity-based vulnerability filtering
- Date-wise vulnerability listing
- Vulnerability summary cards
- Excel report download option
- Responsive UI for desktop and mobile devices
- Sidebar-based navigation
- Clean and consistent design

---

## Technology Stack

- HTML5
- CSS3
- JavaScript
- Chart.js (for vulnerability visualization)
- Bootstrap / Custom CSS

---

## Setup and Installation Instructions

1. Clone the project repository.
2. Navigate to the `frontend` directory.
3. Open `index.html` in a modern web browser.

No external dependencies or build tools are required.
